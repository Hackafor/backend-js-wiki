• Que es JavaScript

    que-es-js
    JavaScript (JS) es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase. Si bien es más conocido como un lenguaje de scripting (secuencias de comandos) para páginas web, y es usado en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat JavaScript es un lenguaje de programación basada en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa

  • Donde se ejecuta Javascript

    ejecutar-js
    JavaScript en un inicio solo se podia ejecutar en los navegadores, pero gracias al proyecto de nodejs en 2011 ahora podemos ejecutar JavaScript del lado del servidor, existen alternativas modernas a nodejs como Deno o Bun pero nodejs es el mas utilizado

  • Que tipos de datos hay en Javascript

    tipos-datos-js
    En JavaScript hay varios tipos de datos. Según el último estándar ECMAScript, existen nueve tipos de datos12. Hay seis tipos de datos primitivos: Undefined (representa una variable que no ha sido declarada o a la cual no se le ha asignado un valor) 3, Boolean (representa un valor lógico y puede tener dos valores: true o false) 3, Number (permite representar y manipular valores numéricos como «37» o «-9.25») 3, String (representa datos textuales o cadenas de caracteres) 3, BigInt (para números enteros de longitud arbitraria) 4 , Null y Symbol, ademas del Object.
    let numero = 8008 let salario = 100_000 let nombre = "Mario Bros" let persona = { nombre: 'Juan', age: 40 } let estado = null let circuloCuadrado = undefined let id = Symbol('identifier')

  • Que es this en Javascript

    this-js
    El valor de this está determinado por cómo se invoca a la función. No puede ser establecida mediante una asignación en tiempo de ejecución, y puede ser diferente cada vez que la función es invocada.
    const test = { prop: 42, func: function() { return this.prop; }, }; console.log(test.func()); // Expected output: 42
